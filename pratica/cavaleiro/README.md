# Cavaleiro Frontend

## Project setup
```
yarn
```

### Compiles and hot-reloads for development
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```

### Run production env local (For PWA features use this command)
```
yarn serve
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
